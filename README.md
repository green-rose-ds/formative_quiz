# formative_quiz
## My Python Quiz for Formative Assessment

###  Welcome to my Python Quiz! It will hopefully be super fun for you.
--------

How the quiz works:

1. You will be asked 4 questions covering fun animal trivia.
2. You will be shown 3 answer options listed as A, B, C
3. Please answer with the letter you think is correct e.g. 'C', you do not need to worry about it being lower or upper case
4. You will be told if your answer is correct and advised of the correct answer if it is incorrect
5. At the end of the quiz you will be shown the correct answers vs. your answers, and given a percentage score

Have fun! 

The quiz questions have come from [Twinkl](https://www.twinkl.co.uk/blog/animal-trivia-questions), where there are many many more interesting questions if you are curious :)
