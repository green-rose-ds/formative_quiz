# formative_quiz
## Documentation to accompany my python code quiz

###  I hope the quiz will be a bit of fun and maybe you will learn some animal trivia!
--------

**How the quiz works:**

1. You will be asked 4 questions in order, which have been written through a tuple.
2. You will be shown 3 answer options listed as A, B, C, also from a tuple.
3. Please answer each question with only the letter you think is correct e.g. 'C', which will be stored to a list and compared to the correct answer (with .strip and lower commands included in the input variable you do not need to worry about your case or adding spaces.
4. You will be told if your answer is correct or incorrect, if your answer is incorrect you will be advised of the correct one using handy f strings (for learning 😸).
5. At the end of the quiz you will be shown the correct answers vs. your answer by making use of operators and functions, and given your score as a percentage again using f strings.

Have fun! 

The quiz questions and answers are sourced from [Twinkl](https://www.twinkl.co.uk/blog/animal-trivia-questions), which has many many more interesting questions if you are curious :)
